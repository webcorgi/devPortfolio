### 폴더구조 및 설명

```
src/
./components => 프레젠테이션 컴포넌트
./containers => 컨테이너 컴포넌트
./json       => 포트폴리오 실제 데이터
./redux      => actions, reducers, states, types
./scss       => variable.scss을 최상위에 등록하고, 하위에서 변수 사용
```
